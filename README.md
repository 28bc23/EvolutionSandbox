# EvolutionSandbox
EvolutionSnadbox is an console aplication with grid.
In the grid are agents (with NeuralNetwork) with food.
Agents get energy when food is eaten.
If agent has 0 energy he dies.
After some time Sandbox evaluates agents and spawn new generation.

## Potencial EvolutionSandbox extension
- Agents can attack each another
- Crossbreeding
- Different types of food
- Different types of agents
- different obstacles than wall (end of grid)
